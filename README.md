Sistema de Gerenciamento de Consultas

Sistema CRUD de um veterinário, onde os funcionários possam adicionar e manipular as consultas. E para os clientes, eles podem marcar uma consulta e acompanha-la. 

Para este projeto final do 2° Semestre do curso Desenvolvimento de Sistemas, estamos utilizando para o Front-end o CSS, HTML e um pouco de JavaScript para algumas funcionalidades integrando ele ao FrameWork Flask do Python. Para o Back-end usamos o Python.
